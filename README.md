
# Python Zahlenratespiel

# Projektbeschreibung:

Dieses Projekt ist ein kleines Python-Konsolenspiel, bei dem der Spieler eine
zufällig generierte Zahl erraten muss. Zu Beginn kann zwischen drei
Schwierigkeitsstufen gewählt werden, die sich im Zahlenbereich und in der Anzahl
der erlaubten Versuche unterscheiden.
Während des Spiels erhält der Spieler nach jedem falschen Versuch einen Hinweis,
ob die eingegebene Zahl zu klein oder zu groß ist. Nach dem Ende einer Spielrunde
(Gewinn oder Verlust) kann sich der Spieler seine bisherigen Ergebnisse anzeigen
lassen, darunter die Anzahl der gewonnenen und verlorenen Spiele sowie die
Gewinnquote in Prozent.
Das Spiel kann anschließend beliebig oft neu gestartet werden.

# Funktionen :

- Auswahl zwischen drei Schwierigkeitsstufen
- Zufällige Generierung einer Zahl
- Begrenzte Anzahl an Versuchen
- Hinweise bei falschen Eingaben (zu klein / zu groß)
- Statistik über gewonnene und verlorene Spiele
- Anzeige der Gewinnquote
- Möglichkeit, das Spiel neu zu starten


# Schwierigkeitsstufen:

| Stufe   | Zahlenbereich | Versuche |
|--------|---------------|----------|
| Einfach | 1 – 10        | 3        |
| Mittel  | 1 – 20        | 4        |
| Schwer  | 1 – 50        | 5        |



# Ausführung:

Start über die Python-Konsole mit der Datei `zahlenratespiel.py`.
